## Junior java developer 👋
Всем привет! Я junior java developer в АО "Гринатом" и учусь на 1-м курсе магистратуры Санкт-Петербургского государственного электротехнического университета «ЛЭТИ». \
За время работы спроектировала и разработала сервис «База знаний» для хранения и обработки JSON-страниц, используемый сотрудниками компании. \
Сервис подят в docker-конейнере и функционирует на сервере, реализующем архитектурный паттерн BFF (Backend For Frontend). \
Проект реализую в стиле TDD (Test-Driven Development, разработка через тестирование). Написание unit-тестов с помощью фреймворка Junit. \
Работаю в команде с четырьмя java-разработчиками в системе контроля версий git. \
Знаю и использую в своей работе стек технологий: Java 21, Spring Boot, PostgreSQL, Docker, SQL, Flyway.


В свободное время изучаю машинное обучение в стеке: Python, Jupyter Notebook.

Окончила бакалавриат РГПУ им. Герцена по специальности "прикладная математика и информатика" математического факультета, который дал мне достаточно обширную базу в области математики и программирования.\
В рамках обучения мной было изучено порядка 18-ти математических дисциплин, в том числе: линейная алгебра, теория вероятностей, математическая статистика, математический анализ, математическое моделирование и другие. \
Также были получены знания и навыки в области машинного обучения, анализа данных, алгоритмов и структурах данных. Были на базовом уровне освоены python и C#.

Регулярно принимаю участие в образовательных программах:\
Ещё на 3-м курсе бакалавриата, в 2023-м году, успешно прошла программу предстажировки в качестве 1С-разработчика от АО "Гринатом".\
После чего была трудоустроена в АО "Гринатом" в качестве Junior Java-разработчика.\
В 2024-м году прошла bootcamp по Си от Сбербанка.\
